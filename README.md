# \# Loan Approval Prediction Project

# 

# This project predicts whether a loan application will be approved or not using machine learning.

# 

# \## Problem

# Banks need to evaluate whether applicants are capable of repaying loans.

# 

# \## Dataset

# Kaggle Loan Approval Dataset.

# 

# Features include:

# \- ApplicantIncome

# \- CoapplicantIncome

# \- LoanAmount

# \- Loan\_Amount\_Term

# \- Credit\_History

# \- Gender

# \- Married

# \- Education

# \- Self\_Employed

# \- Property\_Area

# 

# Target variable:

# Loan\_Status

# 

# \## Workflow

# 1\. Exploratory Data Analysis

# 2\. Data Preprocessing

# 3\. Feature Engineering

# 4\. Feature Selection

# 5\. Model Training

# 6\. Model Evaluation

# 7\. Deployment using Streamlit

# 

# \## Model

# Logistic Regression

# 

# Evaluation Metric:

# ROC-AUC

# 

# \## Deployment

# 

# The machine learning model is deployed using Streamlit to provide an interactive web interface for predicting loan approval.
